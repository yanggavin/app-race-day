# Race Day – Privacy Policy Site

This repository hosts the static Privacy Policy for the Race Day app.

## Local Preview

You can open `docs/index.html` directly in a browser, or serve it locally:

```bash
npx serve docs
```

## Publish via GitHub Pages

To host the policy at `https://yanggavin.github.io/app-race-day/`:

1. Push this repository to GitHub: `https://github.com/yanggavin/app-race-day`
2. In GitHub, go to Settings → Pages
3. Set **Source** to `Deploy from a branch`
4. Set **Branch** to `main` (or default) and **Folder** to `/docs`
5. Save. Your site will be available at the Pages URL

## Updating the Policy

Edit `docs/index.html` and update the "Last updated" date.
